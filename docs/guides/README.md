# OhMyUniversity! - Guides Overview

This section contains practical guides for working with data sources, datasets, and normalization processes used in OhMyUniversity!.

Unlike `getting-started/`, which introduces the project, and `architecture/`, which describes the system design, this section focuses on **how to work with real-world data in practice**.

### What you will find here

- Data source explanations
- Normalization rules
- Data cleaning strategies
- Mapping between systems
- Common data issues and solutions
